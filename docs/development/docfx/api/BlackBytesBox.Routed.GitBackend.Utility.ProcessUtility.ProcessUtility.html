ExecuteProcess(string, string, string)

Executes a process with the specified filename, arguments, and working directory.

Declaration
public static (string Output, string Error, int ExitCode) ExecuteProcess(string fileName, string arguments, string workingDirectory)
Parameters
Type Name Description
string fileName

The executable to run.
string arguments

The command-line arguments for the process.
string workingDirectory

The directory in which to execute the process.
Returns
Type Description
(string Output, string Error, int ExitCode)

A tuple containing the standard output, standard error, and exit code.
Remarks

This method runs the specified process without shell execution, captures both standard output and error streams, and returns these outputs along with the process exit code.

Examples

var result = ExecuteProcess("git", "-C "C:\gitlocal" clone http://localhostuser/gitrepos/MyProject.git", @"C:\gitlocal"); Console.WriteLine($"Output: {result.Output}"); Console.WriteLine($"Error: {result.Error}"); Console.WriteLine($"Exit Code: {result.ExitCode}");

LocateExecutable(string)

Locates an executable in the system PATH.

Declaration
public static (string Directory, string FileName, string FullPath)? LocateExecutable(string fileName)
Parameters
Type Name Description
string fileName

The name of the executable file (e.g., "git" or "git.exe").
Returns
Type Description
(string Directory, string FileName, string FullPath)?

A tuple containing:

  • Directory: The directory in which the executable was found.
  • FileName: The filename with the proper extension.
  • FullPath: The fully qualified path to the executable. Returns null if the executable is not found.
Remarks

On Windows, if the provided filename lacks an extension, ".exe" is appended for the search. This method returns a tuple with the directory where the executable was found, the filename (with extension), and the fully qualified path to the executable.

Examples

var result = LocateExecutable("git"); if (result != null) { Console.WriteLine($"Directory: {result.Value.Directory}"); Console.WriteLine($"FileName: {result.Value.FileName}"); Console.WriteLine($"FullPath: {result.Value.FullPath}"); } else { Console.WriteLine("Executable not found in the system PATH."); }

Parameters

string startDirectory

The root directory to start the search from.
Returns
Type Description
(string Directory, string FileName, string FullPath)?

A tuple containing:

  • Directory: The directory in which the executable was found.
  • FileName: The filename with the proper extension.
  • FullPath: The fully qualified path to the executable. Returns null if the executable is not found.
Remarks

On Windows, if the provided filename lacks an extension, ".exe" is appended for the search. This method searches all subdirectories of the specified starting directory for the executable. It returns a tuple with the directory where the executable was found, the filename (with extension), and the fully qualified path to the executable.

Examples

var result = LocateExecutableRecursively("git", @"C:\Program Files"); if (result != null) { Console.WriteLine($"Directory: {result.Value.Directory}"); Console.WriteLine($"FileName: {result.Value.FileName}"); Console.WriteLine($"FullPath: {result.Value.FullPath}"); } else { Console.WriteLine("Executable not found in the specified directory tree."); }
