Class GitBackendMiddleware

Middleware that integrates Git’s smart HTTP backend by executing git-http-backend.exe. It sets up required CGI environment variables and streams request/response data to enable basic Git clone, fetch, and push operations. It supports Basic authentication by calling a delegate for repository-specific credential validation.
